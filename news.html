2021-12-13

An update of the IBSI 2 reference manual is online (version 5). Changes since the last version are: (i) Updated list of participants; (ii) Clarified the intended use of the "mirror" boundary condition (section 2.2); (iii) The 2D tests in phase 2 are now without interpolation (section 5.2); Moved embedding of convolutional filtering in radiomics workflow to section 1.1; (iv) Added several implementation troubleshooting boxes (chapter 4); (v) Clarified units of sigma and lambda parameters of Gabor filters (section 4.5); (vi) Clarified spatial extent of Gabor filter support (section 4.5); (vii) Clarified implementation of the à trous algorithm (section 4.6.2); (viii) Extended description of non-separable wavelets (section 4.6.4); (ix) Added recommendation to not perform image decomposition beyond the first level using separable wavelets (section 4.6.5); (x) Extended description of the Riesz transform (section 4.7); and (xi) Removed chapter 5 Filtering for Radiomics in Practice.

2021-09-16

The fourth version of the reference manual can now be found on arXiv. Important changes include improved documentation of Gabor filters, a reworked section on non-separable wavelets, elucidation of the a trous algorithm, as well as a greatly expanded section on the Riesz transform. The reference manual can be found here: https://arxiv.org/abs/2006.05470
